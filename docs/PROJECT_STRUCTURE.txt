AI/ML ENVIRONMENT - PROJECT STRUCTURE
======================================

Complete AI/ML environment with neural network implementations in Python.
All models use learned parameters through backpropagation - NO rule-based logic.

DIRECTORY STRUCTURE:
--------------------

├── src/                          # Main source code
│   ├── data/                     # Data handling and loading
│   │   ├── dataset.py           # Custom dataset and data loader
│   │   └── augmentation.py      # Data augmentation for neural networks
│   │
│   ├── models/                   # Neural network models
│   │   ├── neural_networks.py   # NN architectures (FFN, CNN, RNN, Transformer, AutoEncoder)

│   │   ├── activations.py       # Activation functions (ReLU, Sigmoid, Tanh, etc.)

│   │
│   ├── training/                 # Training pipeline

│   │   ├── losses.py            # Loss functions (MSE, Cross-Entropy, Focal, etc.)

KEY FEATURES:
-------------

1. NEURAL NETWORK MODELS (All learned, NO rule-based):
   - FeedForward Neural Networks
   - Convolutional Neural Networks (CNN)
   - Recurrent Neural Networks (RNN/LSTM/GRU)
   - Transformer with Self-Attention
   - AutoEncoder for unsupervised learning

2. TRAINING PIPELINE:
   - Supervised learning trainer
   - Unsupervised learning trainer
   - Reinforcement learning trainer
   - Multiple loss functions (Cross-Entropy, MSE, Focal, Contrastive, etc.)
   - Optimizers (Adam, SGD, RMSprop, AdaGrad, AdamW)
   - Callbacks (Early Stopping, Model Checkpoints, LR Schedulers)

3. DATA PROCESSING:
   - Custom dataset and data loader with batching
   - Data augmentation (Noise, Scaling, Mixup, Cutout)
   - Scalers (StandardScaler, MinMaxScaler, RobustScaler)
   - Transformers (PCA, Feature Selection, Polynomial Features)
   - Encoders (Label, OneHot, Feature, Target)

4. EVALUATION:
   - Classification metrics (Accuracy, Precision, Recall, F1, AUC-ROC)
   - Regression metrics (MSE, MAE, R², MAPE, Huber)
   - Cross-validation
   - Confusion matrix
   - Error analysis
   - Model benchmarking

5. UTILITIES:
   - Comprehensive logging system
   - Model saving/loading
   - Configuration management
   - Visualization utilities
   - Helper functions


QUICK START:
------------

1. Install dependencies:
   pip install -r requirements.txt

2. Run the main demo:
   python main.py

3. Run classification example:
   PYTHONPATH=/vercel/sandbox python examples/simple_classification.py

4. Run regression example:
   PYTHONPATH=/vercel/sandbox python examples/regression_example.py


USAGE EXAMPLE:
--------------

from src.models.neural_networks import FeedForwardNN
from src.training.trainer import SupervisedTrainer
from src.models.optimizers import Adam
from src.data.dataset import CustomDataset, DataLoader

# Create model

    input_dim=20,
    hidden_dims=[64, 32],
    output_dim=3,
    activation='relu'
)

# Create trainer
optimizer = Adam(learning_rate=0.001)
trainer = SupervisedTrainer(model, optimizer, loss_fn='cross_entropy')

# Train
history = trainer.fit(train_loader, val_loader, epochs=10)
